# Miracle Desing Tokens
Miracle Design Tokens is design tokens for Miracle Design System, Design tokens is a single source of truth for designers, developers, and all stackholders to ensur consistency across the design system.
## For using on repo
```
npm install
```
```
npm run build
```

## For using on project
```
npm i miracle-design-tokens
```

```
// add this file to use tokens in scss
// For default (light mode):
@import 'miracle-design-tokens/dist/scss/alias/default-tokens.scss';

// For dark-mode:
@import 'miracle-design-tokens/dist/scss/alias/dark-tokens.scss';
```
